# 自动排班工具项目文档

## 一、项目概述

### 1.1 项目背景
本项目旨在开发一个机场地勤人员自动排班工具，解决人工排班效率低、疲劳度评估不准确、公平性难以保证等问题。

### 1.2 项目目标
- 实现岗位-人员资质配置管理
- 支持导入历史排班表分析人员疲劳度
- 基于约束满足算法自动生成排班表
- 支持人工调整和结果导出

### 1.3 应用场景
机场地勤部门每日排班工作，涉及多个航班、多种岗位和多名员工的调度安排。

## 二、业务需求分析

### 2.1 核心业务流程

```
每日排班流程：
1. 输入当天航班信息（航班号、起飞时间、到达时间）
2. 导入前一天排班表 Excel
3. 系统自动分析人员疲劳度
4. 点击"自动排班"生成当日排班表
5. 人工调整排班结果
6. 导出排班表 Excel
```

### 2.3 疲劳度计算规则

疲劳度 = 基础工时权重 × 工作时长 + 夜班惩罚 + 连续工作惩罚

- **基础工时权重**：每工作1小时疲劳度+1
- **夜班惩罚系数**：夜班时段（22:00-06:00）工时×2
- **连续工作惩罚**：连续工作第2天起，每天额外+5疲劳度

### 2.4 排班约束条件

1. 每人每天最大工作时长：12小时
2. 每人每周至少休息1天
3. 只能安排有资质的人员到对应岗位
4. 同一时间不能安排同一人到多个岗位
5. 不能安排状态为病假/休假的人员

### 2.5 排班优先级策略

1. 优先安排疲劳度最低的人员
2. 兼顾公平性（每人工时均衡）
3. 考虑人员夜班能力

### 3.3 核心算法设计

#### 3.3.1 自动排班算法

采用贪心算法 + 约束回溯策略：

```
算法步骤：
1. 收集所有待排班岗位及其需求人数
2. 按岗位优先级排序（如督导优先）
3. 对每个岗位：
   a. 获取该岗位的可胜任人员列表
   b. 过滤掉状态异常（病假/休假）的人员
   c. 过滤掉时间冲突的人员
   d. 按疲劳度排序（疲劳度低的优先）
   e. 选择前N名人员分配到该岗位
   f. 更新人员已分配状态和预计疲劳度
4. 检查是否有岗位无人可排，如有则尝试跨岗位安排
5. 输出排班结果
```

#### 3.3.2 冲突检测算法

```
冲突类型：
1. 时间冲突：同一人在同一时间段被分配到多个岗位
2. 资质冲突：人员没有该岗位的资质
3. 状态冲突：人员状态为病假/休假
4. 工时冲突：人员当天累计工时超过最大值
5. 疲劳度冲突：人员疲劳度超过阈值

检测方法：
- 在分配前检查人员是否有时间冲突
- 在分配后更新人员状态，后续分配时检查
```

#### 4.3.2 疲劳度计算实现

```javascript
function calculateFatigue(staffId, historyRecords) {
    let fatigue = 0;
    const today = new Date();
    
    // 计算最近7天的工作记录
    const recentRecords = historyRecords.filter(record => {
        const recordDate = new Date(record.date);
        return (today - recordDate) / (1000 * 60 * 60 * 24) <= 7;
    });
    
    // 计算基础工时疲劳度
    recentRecords.forEach(record => {
        const hours = parseFloat(record.workHours) || 0;
        fatigue += hours * baseWeight;
        
        // 检查是否为夜班
        const startTime = parseTime(record.startTime);
        const endTime = parseTime(record.endTime);
        if (isNightShift(startTime, endTime)) {
            fatigue += hours * nightPenalty;
        }
    });
    
    // 计算连续工作惩罚
    const consecutiveDays = countConsecutiveWorkingDays(recentRecords);
    if (consecutiveDays > 1) {
        fatigue += (consecutiveDays - 1) * consecutivePenalty;
    }
    
    return fatigue;
}
```

#### 4.3.3 自动排班算法实现

```javascript
function autoSchedule(flights, positions, staff, historyRecords) {
    const schedule = [];
    const assignedStaff = new Map(); // 记录每人已分配的岗位
    
    // 计算所有人当前疲劳度
    const fatigueMap = new Map();
    staff.forEach(s => {
        fatigueMap.set(s.id, calculateFatigue(s.id, historyRecords));
    });
    
    // 遍历所有航班
    flights.forEach(flight => {
        const flightPositions = flight.positions.split(',');
        
        flightPositions.forEach(positionName => {
            const position = positions.find(p => p.name === positionName.trim());
            if (!position) return;
            
            // 获取可胜任人员
            const qualifiedStaff = staff.filter(s => {
                const qualifiedIds = position.qualifiedStaff.split(',');
                return qualifiedIds.includes(String(s.id));
            });
            
            // 过滤状态异常人员
            const availableStaff = qualifiedStaff.filter(s => s.status === '正常');
            
            // 按疲劳度排序
            availableStaff.sort((a, b) => 
                (fatigueMap.get(a.id) || 0) - (fatigueMap.get(b.id) || 0)
            );
            
            // 分配人员
            const assigned = availableStaff[0];
            if (assigned) {
                schedule.push({
                    flightNo: flight.flightNo,
                    position: positionName,
                    staffName: assigned.name,
                    staffId: assigned.id,
                    startTime: flight.startTime,
                    endTime: flight.endTime
                });
                
                // 更新疲劳度
                const hours = calculateHours(flight.startTime, flight.endTime);
                fatigueMap.set(assigned.id, 
                    (fatigueMap.get(assigned.id) || 0) + hours * baseWeight
                );
            }
        });
    });
    
    return schedule;
}
```

1. **Excel格式要求**：导入的Excel文件必须按照指定模板格式填写
2. **人员编号**：人员编号必须唯一，岗位配置中使用编号关联人员
